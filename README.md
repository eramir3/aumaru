# AUMARÚ

## Installation

Install dependencies with npm

```
npm install
```

Run docker

```
docker compose up -d
```

## Tailwind

To compile and build tailwind go to theme folder and run the following command

```
npx run tailwind
```

## To synchronize local env with host env
1. Download the wp-content folder use cPanelFileManager
2. Downdload the sql dump file with phpmyadmin
